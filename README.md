## 👋 About Me

Hi, I’m Kassam! I’m a 🧑‍💻 Robotics & AI Developer, ⚙️ Embedded Systems Specialist, and 🔧 DevOps Infrastructure Engineer passionate about bridging the gap between physical hardware and advanced software intelligence. 

My work spans across:

- 🤖 **Embodied AI & Humanoid Robotics:** Programming and configuring advanced robotic systems (like the Unitree G1 and G1 EDU platforms), focusing on kinematic control and finite state machines.
- ⚙️ **Embedded Systems & Sensor Fusion:** Developing real-time firmware and sensor processing pipelines using Arduino, ESP32, Raspberry Pi, 3D LiDAR, and IMUs.
- 🧠 **Computer Vision & Deep Learning:** Architecting vision workflows and integrating real-time object detection frameworks like YOLO with Vision-Language-Action models.
- 🔧 **DevOps & Infrastructure:** Building production-grade deployment environments, managing isolated multi-robot communication (ROS 2), and containerizing software stacks using Docker.
- 🌐 **Network Systems & Server Management:** Designing robust network topologies, client-server communication architectures, and automated server infrastructure.
- 💻 **Software Automation:** Writing highly optimized codebase solutions in Python, C/C++, Bash scripting, and modern framework loops.

This repository serves as a central hub for my engineering journey—from autonomous mapping frameworks and AI-driven vision models to custom containerized developer environments. I’m always eager to collaborate on deep-tech breakthroughs, so feel free to reach out! 🤝
